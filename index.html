Projects

Most of these projects were initiated on my own accord and were done as personal projects to both practice my skills as well as to learn new things.

  • blog2

    2017-Present

    A simple social networking platform written using the full *AMP (Apache, MySQL, PHP) stack. It has simple features like posting (with Markdown support), post categories, user profiles, and following. It's due for some major refactoring before I release the source code on GitHub.

  • SoundFace

    2017, UniHack

    An app that uses Microsoft Azure's facial recognition services and the Spotify APIs to generate a mood-based playlist based on your facial expression.

  • Hear Ye

    2017-Present

    It grabs headlines from news sites and literally reads the headlines to you.

  • Antorca

    2016-2017

    Successor to illume OS, but now 64-bit only and with more emphasis on security, access to third-party browsers, and simplicity.

  • Webitor

    2016-Present

    An experimental text editor built with JavaScript and browser local storage.

  • illume OS

    2011-2016, continued as Antorca

    Debian-based Linux distribution with speed, hardware compatibility, and easy access to web services in mind.
