I'm Clarence. I'm a Software Engineering student at Monash University (Clayton campus). Building software that empowers people to achieve their goals is what I've wanted to do since I was 10. If I'm not manifesting these dreams at uni or in assignments, I'm always working on at least one side project.

  • blog2

    2017-2018

    A simple social networking platform written using the full *AMP (Apache, MySQL, PHP) stack. It has simple features like posting (with Markdown support), post categories, user profiles, and following.

  • SoundFace

    2017

    An app that uses Microsoft Azure's facial recognition services and the Spotify APIs to generate a mood-based playlist based on your facial expression. Group project at UniHack 2017, a hackathon organised by Wired at Monash University.

    2017, 2018

    It grabs headlines from news sites and literally reads the headlines to you.
